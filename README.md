# 🎮 RPG com Gameplay Ability System (Unreal Engine)

Este projeto tem como objetivo o desenvolvimento de um RPG completo utilizando a Unreal Engine, com foco em arquitetura de código profissional, mecânicas multiplayer e uso avançado do Gameplay Ability System (GAS).

---

## 🚀 Objetivo do Projeto

Consolidar conhecimentos em desenvolvimento de jogos utilizando **C++** e **Blueprints** na Unreal Engine, criando uma base sólida e escalável para um RPG com sistemas completos de gameplay.

---

## 🧩 Funcionalidades Implementadas

- Sistema de combate  
- Sistema de habilidades (Gameplay Ability System)  
- Sistema de atributos (vida, mana, etc.)  
- Progressão de personagem (XP e Level Up)  
- Inimigos com comportamento básico  
- Estrutura para habilidades e magias  
- Sistema de salvamento *(em desenvolvimento)*  
- Base para multiplayer  

---

## 🧠 Conceitos Aplicados

- Princípios **SOLID**  
- Arquitetura modular e escalável  
- Separação entre lógica em **C++** e **Blueprints**  
- Boas práticas de organização de código  
- Estrutura pensada para projetos de médio/grande porte  

---

## ⚙️ Tecnologias Utilizadas

- Unreal Engine  
- C++  
- Blueprints  
- Gameplay Ability System (GAS)  

---

## 🛠️ Aprendizados Durante o Projeto

- Implementação prática do GAS  
- Estruturação de sistemas reutilizáveis  
- Decisão entre performance (**C++**) vs produtividade (**Blueprints**)  
- Debugging dentro da Unreal Engine  
- Organização de código para manutenção e expansão
