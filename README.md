# 🎮 RPG with Gameplay Ability System (Unreal Engine)

> 🇧🇷 [Read in Portuguese](#-versão-em-português)

This project focuses on developing a complete RPG using Unreal Engine, with an emphasis on professional code architecture, multiplayer mechanics, and advanced usage of the Gameplay Ability System (GAS).

---

## 🚀 Project Goal

The goal is to strengthen my game development skills using **C++** and **Blueprints** in Unreal Engine, building a solid and scalable foundation for a full-featured RPG system.

---

## 🧩 Implemented Features

- Combat system  
- Ability system (Gameplay Ability System)  
- Attribute system (health, mana, etc.)  
- Character progression (XP and Level Up)  
- Basic enemy behavior  
- Structure for abilities and spells  
- Save system *(in progress)*  
- Multiplayer-ready foundation  

---

## 🧠 Concepts Applied

- **SOLID** principles  
- Modular and scalable architecture  
- Clear separation between **C++** and **Blueprints** logic  
- Clean code and organization best practices  
- Structure designed for mid to large-scale projects  

---

## ⚙️ Technologies Used

- Unreal Engine  
- C++  
- Blueprints  
- Gameplay Ability System (GAS)  

---

## 🛠️ What I’ve Learned

- Practical implementation of GAS  
- Designing reusable systems  
- Making decisions between performance (**C++**) and productivity (**Blueprints**)  
- Debugging within Unreal Engine  
- Structuring code for maintainability and scalability

Este projeto tem como objetivo o desenvolvimento de um RPG completo utilizando a Unreal Engine, com foco em arquitetura de código profissional, mecânicas multiplayer e uso avançado do Gameplay Ability System (GAS).

---

## 🚀 Objetivo do Projeto

Consolidar conhecimentos em desenvolvimento de jogos utilizando **C++** e **Blueprints** na Unreal Engine, criando uma base sólida e escalável para um RPG com sistemas completos de gameplay.

---

- Sistema de combate  
- Sistema de habilidades (Gameplay Ability System)

- Sistema de salvamento *(em desenvolvimento)*  
- Base para multiplayer  

---
